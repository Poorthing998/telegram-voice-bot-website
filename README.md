# VoiceText Website

Landing page for VoiceText app.

## Deploy to Vercel

1. Push to GitHub
2. Connect repo in Vercel
3. Deploy with these settings:
   - Framework Preset: **Other**
   - Output Directory: `.` (root)

## Customize

Replace `YOUR_BOT_USERNAME` with your Telegram bot username (search and replace all instances).

## Add Desktop Download

After building your installer, add `VoiceText-Setup.exe` to this folder.
