# Voxly Landing Page

Voice-to-text AI landing page with modern design and animations.

## Deployment on Vercel

1. Upload this folder to a Git repository (GitHub, GitLab, or Bitbucket)
2. Go to [vercel.com](https://vercel.com)
3. Click "New Project"
4. Import your repository
5. Vercel will auto-detect the configuration
6. Click "Deploy"

## Local Development

Simply open `index.html` in your browser to preview locally.

## Features

- Responsive design
- Smooth animations
- Modern glassmorphism UI
- Dark theme with gradient accents
- Optimized for web and mobile

## Tech Stack

- HTML5
- Tailwind CSS (CDN)
- Iconify Icons
- Google Fonts (Outfit & Syne)
